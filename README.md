# Prosox0x0 Security Blog

Personal cybersecurity blog and technical portfolio for Prosox0x0.

## Focus areas

- Penetration testing
- Security research
- Bug bounty
- Reverse engineering
- Malware analysis
- SOC and DFIR
- Red team operations
- CTFs and technical writeups

## Local development

```bash
pnpm install
pnpm dev
```

The local site is available at `http://127.0.0.1:3000/`.

## Production build

```bash
pnpm build
```

The static output is generated in `dist/` and is suitable for GitHub Pages.
